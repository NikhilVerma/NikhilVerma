# Hi, I'm Nikhil

I'm CTO & co-founder of [The Non-Functional Co.](https://nonfx.com/), where we build AI compliance infrastructure for regulated finance. Most of my time goes into a stubborn problem: making LLM agents reliable and deterministic enough to trust in places where mistakes have real consequences.

Before this I spent 8 years at [Bumble](https://bumble.com/) in London, going from engineer to Frontend Core Team Lead across platforms serving 50M+ users — with earlier stops at Opera Software in Sweden and AOL in Bangalore. Nineteen years in, the field still finds new ways to make me feel like a beginner. I write about what I learn along the way.

**Research:** [Active Context Compression: Autonomous Memory Management in LLM Agents](https://arxiv.org/pdf/2601.07190) — an agent that autonomously compresses its own context, inspired by slime-mold exploration. 22% token reduction on SWE-bench Lite with no loss of accuracy.
